Fountain of Dreams - Intro

Introduction

Fountain of Dreams is a 1990 role-playing video game developed and published by Electronic Arts for MS-DOS as a successor to 1988's Wasteland. The game was originally intended as a follow-up to Wasteland, but neither Interplay nor any of the creative team that created Wasteland worked on it. In effect, the game engine is similar, but was created from scratch, and in 2003, Electronic Arts dropped all claims that the game had any connection to Wasteland. Fountain of Dreams was part of the beginning of a trend at EA to produce in-house sequels to its previous titles.

Credits

The game was written and programmed by Dave Albert and Banjo Bob Hardy. Graphics and artwork were done by Peggy A. Brennan, Cynthia Hamilton, and Alan J. Murphy.

Platforms

The only known platform is DOS. The game requires MS DOS 3.0 and higher and supported EGA, MCGA, Tandy / PCJr, and VGA graphics. However, the game did not utilize many advanced features of the VGA controller. All images/graphics are limited to a set of 16 colors.

Packaging

The game came on two 5.25" 360 KB floppy disks. There doesn't seem to be any copy protection. The game should be installed on a hard drive, although I'm not sure if this was a requirement.

Development

Based on the release date and some initial analysis of the game executables it appears that Microsoft C 6.0 (1989) was used to build this game. This compiler used the Microsoft Run-Time Library (RTL) which handled stack checks and exception handling in DOS-based applications. The presence of RTL error codes are visible in the game executables.

Most functions appear to follow cdecl calling convention, although there appears to be some routines that perhaps used inline assembly or use some custom calling convention not commonly used by C.
